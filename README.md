# 🎮 Steam Success Predictor

**Machine Learning модель для предсказания успеха игр на платформе Steam**

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)
![CatBoost](https://img.shields.io/badge/CatBoost-1.2+-green.svg)
![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red.svg)

---

## 📊 О проекте

Этот проект предсказывает, станет ли игра **Хитом** или **Провалом** на основе данных, доступных **до релиза**.

- **Датасет:** 122,611 игр с платформы Steam
- **Модель:** CatBoost (GPU-accelerated)
- **Метрики:** AUC 0.93, Recall 81%, F1 0.52
- **Особенность:** Нет утечки данных (только пред-релизные признаки)

## 📊 Анализ данных (SHAP)

SHAP анализ показывает **почему** модель принимает решения:

- **Категории** и **Цена** — самые влиятельные факторы успеха
- **Снижение цены** обычно увеличивает шанс успеха
- **Количество достижений** положительно влияет на вовлечённость

Графики SHAP сохранены в папке `reports/shap_outputs/`:
- `01_shap_importance.png` — важность признаков
- `02_shap_beeswarm.png` — направление влияния
- `03_shap_hit_example.png` — пример успешной игры
- `04_shap_flop_example.png` — пример провала

## 🎯 Бизнес-ценность

- **Фильтрует 85% провалов** при сохранении 81% хитов
- Может использоваться как **первичный фильтр** для издателей игр
- **Нет утечки данных** — все признаки доступны до релиза
- **Интерпретируемость** через SHAP — можно объяснить решения

## 🔍 Предотвращение утечки данных

**Удалённые признаки (пост-релизные данные):**
- ❌ `Tags` — добавляются пользователями после релиза
- ❌ `Recommendations` — только популярные игры получают рекомендации
- ❌ `Average playtime` — накапливается со временем
- ❌ `Peak CCU` — метрика популярности после релиза
- ❌ `Metacritic score` — рецензии появляются после выхода
- ❌ `User score` — оценки пользователей после игры

**Сохранённые признаки (до релиза):**
- ✅ Цена, достижения, возрастной рейтинг
- ✅ Жанр, категории, платформы
- ✅ Разработчик, издатель
- ✅ Дата релиза
